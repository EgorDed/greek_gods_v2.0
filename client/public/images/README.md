# Фоновое изображение для графа

Чтобы использовать **свою картинку** звёздного неба/туманности вместо стандартной:

1. Положите файл с именем `space-bg.jpg` (или `space-bg.webp`) в эту папку.
2. В коде графа будет использоваться локальное изображение вместо фото с Unsplash.

Рекомендуемое разрешение: от 1920×1080. Подойдут бесплатные фото с [Unsplash](https://unsplash.com/s/photos/nebula), [Pixabay](https://pixabay.com/images/search/nebula/), [Pexels](https://www.pexels.com/search/space/).
